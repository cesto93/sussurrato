# Sussurrato — Agent guide

Minimal Android Jetpack Compose project (single `:app` module).

## Quick reference

```bash
./gradlew :app:assembleDebug          # build APK
./gradlew test                        # run unit tests
./gradlew :app:connectedDebugAndroidTest  # run instrumented tests (device/emulator)
```

## Local dev environment

- **OS**: Linux (Ubuntu) — no Android emulator, uses **Waydroid** for deployment.
- **Java**: JDK 25 (via system `java`). The Gradle wrapper (`./gradlew`) handles the rest.
- **SDK**: `/home/pier/Android/Sdk` (set in `local.properties`, generated by Android Studio).
- **Build/deploy/run flow**: `make build` → `make install` → `make run`. The Makefile sets `ANDROID_HOME` and uses `waydroid` commands.
- **Waydroid session**: `make waydroid-session` starts the session if not already running.
- **App package name**: `com.pierfrancescocontino.sussurrato` (AndroidManifest + Gradle).
- **Entrypoint**: `app/src/main/java/com/pierfrancescocontino/sussurrato/MainActivity.kt` — `ComponentActivity` with Jetpack Compose and `enableEdgeToEdge()`.

## Project structure

- **Gradle 9.4.1** via wrapper, **AGP 9.2.1**, **Kotlin 2.2.10** with Compose compiler plugin.
- **Min SDK 28, target/compile SDK 36.** Java 11 source/target compatibility.
- **Jetpack Compose + Material3** using BOM `2026.02.01`.
- Version catalog: `gradle/libs.versions.toml`.
- **No CI, no linter/formatter config, no README.** Template-level project — almost no custom code yet.

## Conventions

- Kotlin code style: `official` (set in `gradle.properties`).
- Namespace: `com.pierfrancescocontino.sussurrato`. New code should live under `app/src/main/java/com/pierfrancescocontino/sussurrato/`.
- Tests: JUnit 4 for unit tests, `AndroidJUnitRunner` for instrumented tests. Both are placeholder-only.
